# **Cliff Note Builder**
---
## **REQUIRED MODULES**

* textract
* docx2txt
* unicodedata
* docx
For example:
```
# Code block
import textract;
import docx2txt;
from unicodedata import *;
import docx;
from docx import Document;
```
The Cliff Note Builder is made to condense large documents into smaller documents so you can read about your chosen keyword easier.
